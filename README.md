## Description

Built with <a href="https://nestjs.com/" target="blank">Nest.js</a> framework for backend and <a href="https://reactjs.org/" target="blank">React</a> for frontend.

## Installation

```bash
$ npm install
$ cd client && npm install
```

## Running the app

```bash
# concurrently
$ npm run dev

# only backend
$ npm run start:dev

# only frontend
$ npm run dev --prefix client

# only frontend (other way)
$ cd client
$ npm run dev

```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
